### Просмотр информации о статусе линков HBA  - бывает полезным для быстрого понимания ситуации

<details><summary>systool -c fc_host -v | grep </summary>
<p>

```bash
[root@fab0 ~]# systool -c fc_host -v | grep port_state
    port_state          = "Online"
    port_state          = "Linkdown"
```
</p>
</details>
