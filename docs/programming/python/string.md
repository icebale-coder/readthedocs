# Python работа со строками 

## Присваивание значения

```py
	a = 'text1'
	b = "text1"

	print(a)
	tes't

	print(b)
	tes't
```

## Экранирование символов/спец. символы

```py
	a = 'tes\'t'
	b = "tes't"

	print(a)
	tes't

	print(b)
	tes't

```

```py

	a="tes\"t"
	b='tes"t'

	print(a)
	tes"t

	print(b)
	tes"t

```
