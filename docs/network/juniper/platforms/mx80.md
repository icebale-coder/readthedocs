# Juniper MX80

## Настройка netflow v5 

### Настройка секции forwarding-options
#### 10.10.10.10 - адрес flow коллектора
#### 1.1.1.1 - адрес источника, от которого идет netflow на коллектор (обычно указывается loopback устройства)

```bash
forwarding-options {
    sampling {
        sample-once;
        input {
            rate 1;
            run-length 0;
        }
        family inet {
            output {
                flow-server 10.10.10.10 {
                    port 9996;
                    source-address 1.1.1.1;
                    version 5;
                }
            }
        }
    }
}
```

### Настройка секции firewall
```bash
firewall {
    family inet {
        filter SAMPLE {
            interface-specific;
            term default {
                then {
                    sample;
                    accept;
                }
            }
        }
    }
}
```
### Настройка интерфейса, с которого нужно снимать netflow

```bash
interfaces {
	ae0 {
		unit 10 {
            description "<< Subinterface for netflow accounting>>";
            vlan-id 10;
            family inet {
                filter {
                    input SAMPLE;
                    output SAMPLE;
                }
                address x.x.x.x/31;
            }
        }
    }
}
```
